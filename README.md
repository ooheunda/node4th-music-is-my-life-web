## 프로젝트 소개
Spartify  
웹개발 AtoZ 강의 기반의 모두를 위한 음악 추천 사이트  

개발 기간 : 2023.12.22 ~ 2023.12.28 프로젝트 기간 끝

## 구현 기능
* 홈
    * 이미지 무한 롤링 배너
    * 유튜브 API를 활용한 음악 추천 카드
      * 추천 이유 - 더보기 토글
      * 좋아요 버튼
    * 버튼에 사용자 상호작용 기능 구현
      * 작성하기 버튼 - 마우스를 올리면 입체적으로 회전
      * 카테고리 버튼 - 스크롤 시에도 위치 고정
    * 카테고리별 카드 재정렬
* 작성
    * 모달창 구현
    * Firebase로 데이터 연동
    * 배너 이미지 추가 기능
